# crs-full
CRS-FULL é um medidor acústico independente que extrai parâmetros de fala em tempo real. O sistema captura áudio bruto, analisa o silêncio, hesitação, aceleração e features acústicas, exportando dados estruturados e gráficos visuais. Agnóstico de contexto clínico, funciona em qualquer aplicação de análise acústica. Arquitetura transparente.
